## Simulando um Malware de Captura de Dados Simples em Python

Estudo guiado a partir do bootcamp de cibersegurança oferecido pela DIO.

Neste projeto tive a oportunidade de experimentar, compreender e documentar o funcionamento de malwares simulados com Python, em um ambiente 100% controlado e com fins educacionais.

## Objetivos com a Aprendizagem 

Ao concluir este desafio, fui capaz de:

* Compreender o funcionamento prático de Ransomware.
* Identificar como esses malwares exploram vulnerabilidades e brechas humanas.
* Programar scripts simples em Python simulando ataques reais em ambiente controlado.
* Refletir sobre estratégias de defesa e prevenção contra malwares.
